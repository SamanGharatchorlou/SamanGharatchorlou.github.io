Aim

Take control of the whole map by taking out the other teams. Strategically build up your armies and try to hold continents to increase the number of soldiers you receive at the start of your turn. This game can be played with 2-4 computer players.

Developing This Game

This was my first big project, it was a challenge not just building the core game itself but also creating computer players (referred to AI, even though it’s not really AI). For the AI I had to build a complex string of recursive functions (a function that calls itself), which was quite a challenge, particularly for the attack phase. The AI works on a set of basic rules that it follows giving the impression it knows (well sort of) what it’s doing.

Further Developments

I 'completed' this game about a month before I completed it because I was constantly adding little features here and there. There's still a tonne of stuff I want to add, but haven’t got around to yet, a few of them below:

  • Graphics
    The game currently consists of boxes fighting eachother, which is kind of dull. I want to add actual soldiers with fighting, killing and dying animations to give the game more life.
  • Improved Computer Player
    As mentioned the current AI works on a few basic rules. I'm hoping to make the AI a little more situational so it can make some more complex decisions being more unpredictable.
  • Prediction Algorithm
    To do the above I need to build a prediction algorithm to calculate win/loss probabilities of certain battle situations and I'd like to add something into the game giving the player the ability to calculate the chance of winning a fight, maybe once a turn or trade a soldier for 10 uses etc. I figured out equations to calculate 1v1, 1v2, 2v1 and 2v2 battles already. I haven’t been able to calculate the probability for 3v2 battle yet, I know what probabilities are but not the equation (not that you need it once you have the probabilities). If you know how to do it, let me know.
  • Dice Roll
    Ross wants to see what he has rolled so I'll add some dice animations.

There are even more things I want to add like a death laser, but I’ll stop here before I get too carried away.

Further Developments
  • Target Lock
    Currently the towers shoot at their closest target, this obviously changes rapidly as the creeps move across the map causing the towers to flicker across targets rapidly. I will change this so that the towers target the creep closest to the players base within their range, a more conventional (and better) way to do it I think.
  • Tower Types
    Currently there are 6 waves and 2 tower types each with an upgrade option, 2 enemy types and a boss. I'd like to expand the tower types, including: a short range, rapid fire tower; a slow fire, splash damage tower and a low damage, slowing tower. Expanding the upgrade options so you have multiple options each having various benefits would be an even further development.
  • Enemy Types
    I'll look to expand the enemy rooster eventually, but that'll come last. For now, I'll just tweak them to keep the game balanced.
  • More Waves!
    Before I add more tower and enemy types I think we want more enemy waves to get the game going a bit more.
